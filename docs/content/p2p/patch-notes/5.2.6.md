# P2P v5.2.6

**Release Date:** November 22, 2025

## What's Fixed

### Creating and Editing Offers
- Fixed database errors when creating or editing offers
- Fixed corrupted data issues that caused offers to fail

### Dashboard
- Fixed wallet balance showing $0 for all users
- Dashboard now shows correct total balance from all your wallets
- Now properly converts all currencies to USD for accurate total

### Buy/Sell Button
- Fixed "Transaction cannot be rolled back" error when buying/selling
- Fixed buy button not working - trades are now created successfully
- Buy button now works reliably without errors
- Added better error messages if something goes wrong

### Removing Offers
- Fixed remove offer button - now actually deletes offers when you confirm
- Improved feedback when deleting offers
- Fixed redirect after deletion

### Pausing Offers
- Fixed errors when pausing or editing offers with dynamic pricing
- Pausing offers now works smoothly

### Currency Display
- Fixed price currency always showing USD instead of your selected currency (EUR, GBP, etc.)
- Offer forms now correctly display the currency you selected
- Fixed confusing "cryptocurrency" label when selecting FIAT currencies
- Now shows "Currency" for FIAT and "Asset" for crypto

### Trade Forms
- Fixed "Time Limit60minutes" display - now shows proper spacing
- Fixed buy/sell button being disabled incorrectly
- Fixed min/max amounts showing wrong values
- Fixed duplicate currency codes (was showing "100.00 BTC BTC", now shows "100.00 BTC")

### Wallet Balance
- Fixed $0 balance showing when currency prices are unavailable
- Dashboard now shows accurate balances even if some prices are missing
