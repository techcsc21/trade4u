üé® Minting NFTs

Route: /nft/create

6-Step Minting Wizard:

Step 1: Select Collection

Choose an active, deployed collection you own. Only approved collections appear here.

Step 2: Connect Wallet

Connect wallet and validate network. System checks:

  • ‚Ä¢ Chain ID matches collection blockchain
  • ‚Ä¢ Sufficient balance for gas fees
  • ‚Ä¢ Wallet signature authorization

Step 3: Provide IPFS Image

Three options to provide your NFT image:

  • ‚Ä¢ Upload to platform: Upload image directly (10MB max, PNG/JPG/WebP/GIF)
  • ‚Ä¢ Your IPFS URL: Paste your own IPFS URL (ipfs:// or gateway URL)
  • ‚Ä¢ IPFS guide provided: Step-by-step instructions for Pinata, NFT.Storage, Web3.Storage

System validates:

  • ‚Ä¢ IPFS URL format correctness (CIDv0 and CIDv1 support)
  • ‚Ä¢ Gateway URL conversion (ipfs:// ‚Üí https://)
  • ‚Ä¢ No duplicate metadata URIs (prevents reminting same content)

Step 4: Add Details

  • ‚Ä¢ Name: NFT title (required)
  • ‚Ä¢ Description: Detailed information about the NFT
  • ‚Ä¢ External URL: Link to additional content
  • ‚Ä¢ Tags: Keywords for discoverability

Step 5: Configure Properties

  • ‚Ä¢ Attributes: Key-value pairs (e.g., "Rarity: Legendary", "Color: Blue")
  • ‚Ä¢ Royalties: Secondary sale percentage (0-50%, inherits collection default)
  • ‚Ä¢ Unlockable Content: Private content revealed to owner only

Step 6: Preview & Mint

Final review with:

  • ‚Ä¢ Complete NFT preview
  • ‚Ä¢ Gas fee estimation
  • ‚Ä¢ Affordability check (warns if low balance)
  • ‚Ä¢ One-click mint button

‚úÖ Blockchain Integration

After final review, you sign the mint transaction in your wallet (MetaMask, etc.). The system records the transaction hash, block number, and updates your creator profile. All minting happens directly on the blockchain - you maintain full custody.

üõí Buying & Bidding

Fixed-Price Purchase

  1. 1. Browse marketplace or view NFT details
  2. 2. Click "Buy Now" button
  3. 3. Connect wallet if not connected
  4. 4. System validates:
    • ‚Ä¢ Network matches NFT chain
    • ‚Ä¢ Sufficient balance
    • ‚Ä¢ KYC status (if high-value)
  5. 5. Approve transaction in wallet
  6. 6. Ownership transfers instantly

Auction Bidding

  1. 1. View auction details and current bid
  2. 2. Enter bid amount (must exceed minimum)
  3. 3. Click "Place Bid" button
  4. 4. System checks:
    • ‚Ä¢ Bid meets minimum increment
    • ‚Ä¢ Auction still active
    • ‚Ä¢ Not your own auction
  5. 5. Previous bidder auto-refunded
  6. 6. You become highest bidder
  7. 7. Win NFT when auction ends

Making Offers

Make offers on any NFT (even if not listed):

  • ‚Ä¢ Click "Make Offer" button
  • ‚Ä¢ Enter offer amount and expiration
  • ‚Ä¢ Seller receives notification
  • ‚Ä¢ Seller can accept, reject, or counter
  • ‚Ä¢ Offer expires automatically if not accepted

Note: Offers require marketplace settings to have offers enabled.

Transaction Fees

When purchasing NFTs, you pay:

  • ‚Ä¢ NFT Price: Set by seller
  • ‚Ä¢ Marketplace Fee: % goes to platform (default 2.5%)
  • ‚Ä¢ Creator Royalty: % goes to original creator (0-50%)
  • ‚Ä¢ Gas Fee: Blockchain transaction cost

Gas Optimization: Gas estimates shown before every transaction. Transactions may fail if gas price spikes.
