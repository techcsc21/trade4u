# ICO v5.2.4

**Release Date:** November 22, 2025

## What's Fixed

### Admin Management
- Fixed admin approve/reject/flag/unflag actions showing "Something went wrong!" error
- Fixed token type showing random codes instead of readable names (e.g., "Governance", "Utility")
- Admin actions now work properly and refresh correctly

### Notifications
- Fixed ICO notifications sending to wrong recipient
- Notifications now sent to project owners instead of admins
- Fixed notification links - now correctly open your token page

### Dashboard
- Fixed "Access Denied" error when viewing your ICO transactions
- You can now view your investment history without admin privileges

### User Interface
- Fixed typo: "What isStarfluxcoin?" → "What is Starfluxcoin?"
- Hidden fake activity data before ICO starts - now shows only real activity
- Added helpful tooltips to simulator parameters (Monthly Growth Rate and Volatility)

### Token Economics Simulator
- Expanded Monthly Growth Rate range from -10% to +20% → -50% to +50%
- Expanded Volatility range from 0% to 50% → 0% to 100%
- Now supports modeling extreme market conditions

### Investment
- Fixed "insufficient balance" error when you actually have enough funds
- Fixed broken contact support link on Launch Your Token page
- Added wallet address validation to prevent invalid addresses
- Better error messages showing required vs. available balance

### Admin Controls
- Fixed ICO deletion errors from admin panel
- Improved transaction handling when deleting offerings
